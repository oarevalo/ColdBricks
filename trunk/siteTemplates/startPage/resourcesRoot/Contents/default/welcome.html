Welcome to StartPage!

StartPage is a sample application that shows a simple personalized homepage built on HomePortals. Your start page screen can be customized by adding or removing interactive widgets.

To start customizing your page, you must first login into your page. The default username and password are:

Username: default
Password: default

You can modify the password in ColdBricks using the account management module by editing the properties for the account 'default'.

Have fun!
